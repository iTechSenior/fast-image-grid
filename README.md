# fast-image-grid
React Native, Typescript, Unsplash API

Clone the repository:
```
git clone https://github.com/proIT324/fast-image-grid.git
```

Install packages using Yarn:
```
cd ./fast-image-grid
yarn
```

Install packages using NPM:
```
cd ./fast-image-grid
npm i
```

Run the project:
```
react-native run-ios
```
